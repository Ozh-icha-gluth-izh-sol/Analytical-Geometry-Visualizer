# Geometry Transform Visualizer

Интерактивное приложение на **Python** для визуализации геометрических преобразований фигур на координатной плоскости.

___
## О проекте

**Geometry Transform Visualizer** — это учебный проект, который позволяет наглядно работать с геометрическими преобразованиями фигур в 2D.

Проект объединяет две идеи:

- **интерактивное построение объектов**;
- **визуализацию геометрических преобразований**: переноса, поворота, масштабирования, отражения и т.д.

Возможности пользователя:

- ставить точки мышкой;
- строить по ним фигуру;
- перетаскивать вершины и сразу видеть перестроение;
- видеть координаты точек;
- применять преобразования к фигуре;
- сравнивать исходный и преобразованный объект.

## Основная функциональность

В приложении планируется реализовать:

- координатную плоскость с осями и сеткой;
- создание точек мышкой;
- построение фигуры по заданным точкам;
- перетаскивание вершин мышкой;
- отображение координат точек;
- применение преобразований к фигуре;
- визуальное сравнение исходной и преобразованной фигуры;
- последовательное применение нескольких преобразований.

---

## План по итерациям

### Итерация 1


1. **Отрисовка координатной плоскости**  
   Реализация окна приложения, координатных осей и сетки.

2. **Отрисовка фигуры**  
   Отображение многоугольника по заданным вершинам.

3. **Базовые геометрические преобразования**  
   Реализация переноса, поворота и масштабирования фигуры.

---

### Итерация 2

1. **Перетаскивание точек мышкой**  
   Возможность двигать вершины фигуры и автоматически перестраивать изображение.

2. **Дополнительные геометрические возможности**  
   Отображение координат точек и добавление отражений относительно осей.

3. **Композиция преобразований и улучшение интерфейса**  
   Возможность применять несколько преобразований подряд, добавить кнопку сброса и список выполненных действий.

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Ozh-icha-gluth-izh-sol/Analytical-Geometry-Visualizer.git
cd Analytical-Geometry-Visualizer
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv_scraping
```

### 3. Активировать виртуальное окружение

На macOS/Linux:

```bash
source venv_scraping/bin/activate
```

На Windows:

```bash
venv_scraping\Scripts\activate
```

### 4. Установить зависимости

```bash
pip install -r requirements.txt
```

### 5. Запуск приложения

```bash
python main.py
```

### 6. Запуск тестов
```bash
pytest -v
```

### 7. Запуск профайлинга
```bash
python scripts/profile_project.py
```

**Результат будет лежать в** 

```bash
reports/profiling_report.md
```
### Запуск через Docker образ

```bash
docker build -t geometry-visualizer .
docker run --rm geometry-visualizer
```
